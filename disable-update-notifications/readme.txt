Disable email notifications after wordpress updated

== Description ==

WordPress自3.7版本后内置了自动升级功能，对于安全度的提高非常有帮助。

但是每次升级都会有邮件提醒。特别是开发版本，每天自动升级一次，那么每天都会收到至少两封邮件，太过频繁了。

[More Description](http://blog.leniy.org/disable-notification-emails-after-wordpress-auto-updates.html)

== Installation ==

1. Upload `disable-update-notifications` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
